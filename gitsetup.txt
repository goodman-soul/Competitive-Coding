1. Installing Git->

sudo apt-get install git

creating a github account

2. git config --global user.name "your username"
3. git config --global user.email "your_emain"
4. Go to project folder
	-> git remote add origin https://github.com/<your_username>/<myproject>.git
	-> git add <file_name> 
	or
	->git add . (for all files)
	->git commit -m "your message"
	-> git push origin master

Note: this master can be anything->creating a pull request.

Delete files/folders:-

->git checkout master
->git rm -r <folder name>
or
->git rm <file1><fil2>....
->git commit -m "remove folder"
->git push origin master
